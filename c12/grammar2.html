얼마나 -(으)ㄴ/는지 모르다

문법 설명

어떤 것을 아주 아주 아주 많이 강조하고 싶을 때 사용해요. '정말 ~해요' 라는 뜻이에요.

보통 놀랐을 때나, 감동했을 때의 기분을 표현할 때 자주 써요.

예시

🗣️ "요가가 자세 교정에 얼마나 좋은지 몰라요."

🗣️ "어제 이사해서 몸이 얼마나 피곤한지 몰라요."

🗣️ "김치가 우리 몸에 얼마나 좋은지 몰라요."

자세히 알아보기 (형태)

형용사 (Adjective)

  • 받침이 없거나 'ㄹ' 받침: -ㄴ지 모르다 (예: 예쁘다 → 얼마나 예쁜지 몰라요)
  • 받침이 있을 때: -은지 모르다 (예: 좋다 → 얼마나 좋은지 몰라요)

동사 (Verb)

  • 항상: -는지 모르다 (예: 먹다 → 얼마나 먹는지 몰라요)

과거 (Past Tense)

  • -았/었는지 모르다 (예: 힘들었다 → 얼마나 힘들었는지 몰라요)

활동 1: 문장 완성하기 (초급)

괄호 안의 단어를 사용해서 문장을 완성해 보세요.

1. 이 케이크가 (맛있다). → 이 케이크가 얼마나 맛있는지 몰라요.

2. 어제 본 영화가 (재미있다). → 어제 본 영화가 얼마나 재미있는지 몰라요.

3. 한국어 공부가 (어렵다). →
4. 친구가 선물을 받고 (좋아하다). →

정답

3. 한국어 공부가 얼마나 어려운지 몰라요.

4. 친구가 선물을 받고 얼마나 좋아하는지 몰라요.

활동 2: 친구에게 추천하기 (응용)

'얼마나 -(으)ㄴ/는지 모르다'를 사용해서 친구에게 여러분이 좋아하는 것을 추천해 보세요.

보기

A: 요즘 무슨 노래 들어요?

B: BTS 노래를 들어요. 그 노래가 얼마나 좋은지 몰라요.

이제 여러분이 직접 해보세요!

활동 3: 경험 이야기하기 (복습)

최근에 아주 놀랐거나, 아주 좋았거나, 아주 힘들었던 경험에 대해 이야기해 보세요.

보기

"지난 주말에 제주도에 갔어요. 바다가 얼마나 아름다웠는지 몰라요."

"어제 시험공부를 했어요. 얼마나 피곤했는지 몰라요. 그래서 오늘 늦게 일어났어요."

여러분의 경험은?

아래 주제로 글을 써봐도 좋아요:

  • 최근에 재미있게 본 영화/책
  • 최근에 갔던 여행
  • 가장 기억에 남는 식사
