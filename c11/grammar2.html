한국어 문법 마스터 🚀

A/V-(ㄴ/는)다면, N(이)라면

언제 사용할까요?

'-(ㄴ/는)다면'은 실제로 일어나지 않은 일이나, 일어날 가능성이 아주 낮은 일을 상상하거나 가정할 때 사용해요. "만약 ~하면...?" 이라고 말하고 싶을 때 쓰는 표현이에요.

어떻게 만들까요?

동사 (Verbs)

  • 받침 있으면 + 는다면: 먹다 → 먹는다면
  • 받침 없으면 + ㄴ다면: 가다 → 간다면

형용사 (Adjectives)

  • 받침과 상관없이 + 다면: 예쁘다 → 예쁘다면, 작다 → 작다면

명사 (Nouns)

  • 받침 있으면 + 이라면: 학생 → 학생이라면
  • 받침 없으면 + 라면: 의사 → 의사라면

예문 살펴보기

만약 제가 부자라면 세계 여행을 다닐 거예요.

만약 복권에 당첨된다면 좋은 차를 살 거예요.

시간이 더 많다면, 한국어 공부를 더 할 텐데요.

만일 한국어를 배우지 않았다면 한국 회사에 취직하지 못했을 거예요.
