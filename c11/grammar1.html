한국어 사동사(使動詞) 학습

Causative Verbs Practice

사동사란 무엇일까요? 🤔

사동사는 문장의 주체가 다른 대상에게 어떤 행동이나 동작을 하도록 '시키는' 또는 '만드는' 것을 나타내는 동사입니다. 영어의 "make", "let", "have"와 비슷한 역할을 합니다.
예를 들어, '아이가 옷을 입다'는 아이 스스로 하는 행동이지만, '엄마가 아이에게 옷을 입히다'는 엄마가 아이에게 옷을 입도록 시키는 행동을 의미합니다.

사동사는 어떻게 만드나요? 🛠️

일반 동사(능동사)의 어간에 접미사 -이-, -히-, -리-, -기-, -우-, -구-, -추- 등을 붙여서 만듭니다. 어떤 접미사가 붙는지는 동사마다 다르기 때문에, 익숙해지는 것이 중요합니다.

① -이-

  • 먹다 → 먹이다
  • 보다 → 보이다
  • 죽다 → 죽이다
  • 속다 → 속이다

② -히-

  • 입다 → 입히다
  • 앉다 → 앉히다
  • 읽다 → 읽히다
  • 넓다 → 넓히다

③ -리-

  • 살다 → 살리다
  • 알다 → 알리다
  • 울다 → 울리다
  • 놀다 → 놀리다

④ -기-

  • 신다 → 신기다
  • 웃다 → 웃기다
  • 맡다 → 맡기다
  • 벗다 → 벗기다

⑤ -우- / -구- / -추-

  • 자다 → 재우다
  • 서다 → 세우다
  • 타다 → 태우다
  • 맞다 → 맞추다

연습 문제 ✍️
